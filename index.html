Arkhadian (ARKH)

ARKH is a blockchain built using Cosmos SDK and Tendermint and created with Starport. The purpose of $ARKH token is enormous, made for DAO creation, transactional purposes and spendings, built on the cosmos Blockchain.

The $ARKH token relies on CosmWasm, a framework that helps developers to create secure smart contract.

$ARKH Use Cases

Due to the transparent nature of public blockchains, there has never been a protocol that allows for complete or central control over data transparency.

Decentralized Finance (DeFi)

The $ARKH token offers you the opportunity to interact on the Blockchain and make transactions without any bank or brokerage involved.

DAO creation

The Arkhadian Project makes room for DAO creation, the entire infrastructure of a DAO blockchain platform is solely managed by its members, who collaborate on key project decisions, including technological improvements and treasury allocations.

Protocol DAOs

ARKH token is not left behind on this, as we have chosen the cosmos ecosystem for this purpose and to enlarge our horizon, more would be done to sustain this course!.

Payments And Transactions

The Arkhadian Project spend your cryptocurrencies and stablecoins without revealing wallet balances and worst of all, your entire transaction history.

Staking

Staking rewards are generated and distributed to staked $ARKH holders. These holders receive earnings in percentage based on the validator used.

Frequently Asked Question

1What is the primary objective of Arkhadian?

The primary objective of Arkhadian is to create a blockchain network that can operate smoothly and seamlessly even if one node is down.

2 On What Blockchain is $ARKH built on?

Built on the cosmos Blockchain, the $ARKH token relies on CosmWasm, a framework that helps developers to create secure smart contract.

3How can I become an $ARKH Validator?

To become a validator, start by signing and submitting a BondTx transaction and making a request to become a validator on our discord platform.

4 Does Arkhadian have any use case?

Yes! The use cases are:
1. Decentralized finance (DeFi)
2. DAO creation
3. Payment and transactions
